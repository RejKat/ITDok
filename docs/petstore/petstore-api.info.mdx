Ez az API dokumentáció a Petstore rendszer REST végpontjait mutatja be.

A dokumentációban megtalálod az alábbiakat:

- Állatok lekérdezése (GET /pets)
- Új állat rögzítése (POST /pets)
- Válaszstruktúrák és státuszkódok
- Tesztelhető paraméterek és sémák
