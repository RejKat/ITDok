# List all pets

Ez az API-végpont visszaadja az adatbázisban szereplő összes háziállatot.

## Végpont

- **HTTP metódus**: `GET`
- **URL**: `/pets`

## Válasz (200 OK)

```json
[
  {
    "id": 1,
    "name": "Buksi",
    "type": "kutya"
  },
  {
    "id": 2,
    "name": "Cirmos",
    "type": "macska"
  }
]
